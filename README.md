## Helps you find what to eat for the day, night, or morning.

### What it does right now:    
Asks you whether you want to eat inside or outside, depending on it will ask more questions about what cuisine, dietary restrictions, preference, etc. to come up with a dish for you to make with proper ingredients list and the recipe.    
Currently only the inside mode works, outside mode is buggy and will require a full overhaul later.
